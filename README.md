# joaorca.github.io

Currículo/portfólio pessoal de João R C Almeida, hospedado no GitHub Pages.

## Como usar

Edite `index.html` diretamente para atualizar o conteúdo. Para visualizar localmente:

```bash
python3 -m http.server 8080
```

Acesse `http://localhost:8080`.

## Estrutura

```
index.html        # página principal (seção única)
css/              # Bootstrap (local) + main.css
img/              # avatar, favicon, ícones
CNAME             # domínio customizado
```

## Deploy

Push para `master` publica automaticamente via GitHub Pages.
